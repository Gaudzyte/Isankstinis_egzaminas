# Egzamino išankstinė užduotis
Ši programa nuskaito tekstą iš įvesties failo ir sukuria du išvesties failus: 
1. "rezultatai.txt" išveda žodžius, kurie pasikartojo daugiau nei vieną kartą ir nurodomos eilutės, kuriose žodis pasikartoja;
2. "urlRezultatai.txt" išveda tekste rastas internetines nuorodas.
